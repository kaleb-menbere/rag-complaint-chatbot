# RAG Complaint Chatbot

## Project Overview
This project implements a Retrieval-Augmented Generation (RAG) chatbot
to analyze and respond to customer complaints for CrediTrust Financial.

## Features
- Complaint ingestion and preprocessing
- Vector-based semantic search
- Context-aware chatbot responses
- Designed for financial services feedback analysis

## Tech Stack
- Python
- LangChain
- FAISS
- OpenAI
- FastAPI

## Project Structure
- `data/` – complaint datasets
- `vector_store/` – embeddings storage
- `src/` – core RAG logic
- `app.py` – application entry point
